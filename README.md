# Django Snippets

Aplicación web donde usuarios registrados pueden crear y gestionar *snippets* de código en distintos lenguajes de programación.

---

## Sobre el proyecto

Este repositorio es una prueba técnica basada en Django.

Lo retomé después de haberla intentado completar años atrás, cuando logré hacer una versión parcial pero no pude finalizar el ciclo completo de desarrollo debido a problemas personales.

Con el tiempo fui mejorando mis conocimientos en desarrollo frontend y backend, bases de datos, control de versiones y metodologías de trabajo, lo que me permitió volver a este proyecto como una forma de medir mi progreso y consolidar lo aprendido.

Este proyecto representa tanto una revisión técnica como una validación personal de mi evolución como desarrollador.
